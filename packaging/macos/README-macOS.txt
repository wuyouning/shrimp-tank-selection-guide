Shrimp Tank Selection Guide - macOS Installer

This DMG is designed for both Intel Macs and Apple Silicon Macs.

Included files:
- Install Shrimp Tank.command
- Run Shrimp Tank.command

How to use:
1. Make sure Node.js 22.14 or newer is already installed.
2. Double-click "Install Shrimp Tank.command".
3. After install, double-click "Run Shrimp Tank.command" or use Terminal:
   - shrimp-tank --lang zh-CN
   - openclaw-preflight --lang en

Notes:
- This DMG ships the CLI package and a local installer flow.
- The CLI itself is architecture-neutral JavaScript, so the same DMG works for Intel and Apple Silicon as long as Node.js is installed for that Mac.
